## 📌 Project Overview

This tool is meant to serve as a way to measure your minerals, vitamins, macro and micronutrient intake through meal tracking. It allows users optimise their diet by providing technical analysis against their needs & provides recommendations.

---

## 🛠  Planned Features

* Frequent updates on existing food & meal pricing
* Adding auto meal creations based off micro and macro requirements, price, food exclusions, allergies
* Barcode scanning on phone app
* Logs & Error tracking
* Data set of meals
